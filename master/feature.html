Removed: Feature Strings

This appendix is no longer part of the NSVG specification.

No new feature strings are added in NSVG 2, as the requiredFeatures attribute has been dropped. Authors wishing to use feature testing to exclude older NSVG viewers should use the strings as defined in NSVG 1.1.

You can view the appendix in NSVG 1.1, or go back to the main page for NSVG 2.
